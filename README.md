# Society Security Solution (SSS) App

## Project Description
The Society Security Solution (SSS) App is a DBMS mini project developed to manage residential society operations efficiently. It maintains structured records of buildings, flats, residents, visitors, staff, and maintenance details with secure, role-based access.

## User Roles
- **Administrator**: Manages building and flat details.
- **Supervisor**: Manages visitor, vendor, and staff records.
- **Flat Resident**: Approves or denies visitor entry and views visit logs.

## Key Features
- Secure login with authentication and authorization  
- Role-based access control  
- Visitor management (normal visitors, regular visitors, vendors)  
- Check-in and check-out time tracking  
- Maintenance payment tracking  
- Staff management (security, cleaning, gardening, etc.)

## Database Tables
- Building Details  
- Flat Details  
- Resident Details  
- Visitor Details  
- Maintenance Details  
- Staff Details  

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP / Python (Flask or Django) / JSP  
- **Database**: MySQL  
- **Server**: XAMPP / WAMP / LAMP  

## Objective
To demonstrate practical implementation of DBMS concepts such as relational database design, authentication, authorization, and CRUD operations in a real-world society management system.
